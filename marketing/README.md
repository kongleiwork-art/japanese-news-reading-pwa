## 📁 文件说明

### 1. xiaohongshu-promotion.html
主要的推广素材展示页面，包含：
- ✅ 完整的推广文案（两套风格）
- ✅ App核心功能介绍
- ✅ 5张AI宣传图片（自动加载）
- ✅ 发布建议和标签推荐

### 2. 使用方法

**方法一：直接打开HTML文件**
```bash
# 在浏览器中打开
open marketing/xiaohongshu-promotion.html
# 或者
xdg-open marketing/xiaohongshu-promotion.html
```

**方法二：启动本地服务器**
```bash
cd /workspace/marketing
python3 -m http.server 8080
# 然后访问 http://localhost:8080/xiaohongshu-promotion.html
```

**方法三：复制文案直接使用**
- 打开HTML文件
- 选择需要的文案
- 复制到小红书编辑器
